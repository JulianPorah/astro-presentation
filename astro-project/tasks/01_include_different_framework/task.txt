In Astro kann man verschiedene Frameworks einbinden.
Aufgaben:
1. Erstelle eine neue Vue und Svelte Komponente in dem components Ordner und füge sie in der index.astro
2. Schreibe dazu das attribute client:only="svelte" und dasselbe mit vue
Probiere gerne aus warum wir das attribute hinzufügen

Warum müssen wir client:only benutzen?
1. Überspringt es das rendern auf dem Server und rendert somit auf dem Client
2. Wir müssen den Framework-Namen übergeben, weil Astro nicht
die Komponenten auf dem Server lädt und somit nicht weißt
welches Framework man benutzt, außer man sagt ihm das explicit
